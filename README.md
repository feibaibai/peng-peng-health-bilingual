# 蓬蓬科普 (Peng Peng Health) - 中英文双语版

本项目已成功改造为中英文双语版本，支持一键语言切换，并保留了所有核心功能。

## 主要改进

1.  **多语言架构**：基于 Flask 和 Jinja2 构建了动态翻译系统，通过 `translate` 函数和翻译字典实现内容实时切换。
2.  **英文品牌命名**：根据要求，英文版统一使用 **Peng Peng Health** 作为品牌名称。
3.  **全页面覆盖**：
    *   **首页 (Home)**：完整的欢迎辞和功能介绍翻译。
    *   **用药知识库 (Drug Knowledge Base)**：支持文章标题、分类、标签及内容的双语显示。
    *   **个性化中心 (Personalization Center)**：表单项及动态推荐结果均支持双语。
    *   **专题区 (Special Topics)**：精选专题内容的双语化。
    *   **关于我们 (About Us)**：详细的项目背景、核心技术、数据来源及免责声明的精准翻译。
4.  **语言切换器**：在页面顶部导航栏添加了直观的“中文 | English”切换按钮，状态自动持久化。

## 技术细节

*   **后端**：Python Flask
*   **前端**：HTML5, CSS3 (响应式设计)
*   **翻译机制**：自定义 Jinja2 上下文处理器，支持模板内 `{{ _("文本") }}` 调用。
*   **状态管理**：使用 Flask Session 记录用户语言偏好。
